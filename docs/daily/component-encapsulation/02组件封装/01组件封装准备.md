# 组件封装准备

::: tip 前言
组件封装准备
:::

## 1. 实例内容
封装常见的功能组件（Button，Modal，Form相关），封装完成后封装成UI组件库发布到NPM上。
## 2. 涉及知识点
- vue基础语法
- 组件基本语法
- 字键通讯（sync，provide，inject）
- 插槽使用
- prop校验
- 过渡与动画处理
- 计算属性与监听属性
- v-model语法糖
- vue插件机制
- npm发布
## 3. 实例目的
- 掌握组件封装的语法和技巧
- 学会造轮子，了解组件库实现原理
- 搭建和积累自己的组件库
## 4. 学习前提
- 有一定vue基础，懂vue语法
- 熟悉ES6的一些常见语法
- 对vue感兴趣
